# Xentra Linux 🐧

> Lightweight. Modern. Open for Everyone.

Xentra Linux is a modern Ubuntu-based Linux distribution project developed by RootLabs with a focus on:

* lightweight performance
* user-friendly experience
* application compatibility
* productivity
* gaming
* and accessibility for everyday users.

The project aims to create a Linux experience that is simple, modern, and comfortable for students, developers, low-end devices, and users transitioning from other operating systems.

# Why Xentra?

Many older laptops and Chromebooks still have the potential to become useful devices for:

* education
* productivity
* programming
* media consumption
* and daily activities.

Xentra aims to provide a lightweight and modern Linux experience that helps users get more from their hardware without unnecessary complexity.
